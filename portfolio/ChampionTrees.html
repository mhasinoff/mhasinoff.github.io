Champion Trees Web App

This final project was created by myself and my classmate, Mike Adduci. It involved the front and back end design of a web app. At the front end, the user is able to 1) submit data of an historic tree, tree of interest or (potential) champion tree, which is essentially, the largests specimen of it is kind; 2) access the data of previously submitted trees. On the back end, a database of the submitted trees and trees curated by the develops. Currently, this web app is on a local host and will hopefully be made public in the near future.
